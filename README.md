# Daily GM 🌞

A **fundingless** onchain GM project on Base Chain.

- Say "GM" daily
- Earn $GM tokens
- Build streaks
- No VC. No BS. Just vibes.

## Tech Stack
- Next.js 14 (App Router)
- RainbowKit + Wagmi
- Tailwind CSS
- Vercel (hosting)

## Deploy
1. `npm install`
2. `npm run dev`
3. Deploy to Vercel

Built with ❤️ for the Base community.
